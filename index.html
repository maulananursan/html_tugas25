Mengenal Semantic Element di HTML5

  • Article

    • Ketika ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer, karena tag article memiliki type block, artinya element ini memiliki lebar sesuai dengan lebar browser jika kita tidak mengatur lebarnya

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan indentitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukannya.
